## General

- No oop / inheritance
- No for loop


## Types

```
true
 "hello"
1
1.1
```

## Assignemnt

```
let status = true
let msg = "hello"
let num = 1
let float = 1.1
```

## Functions

```
fn () -> 1

fn (a, b) -> 2

fn hello(a, b)
    let a = 5
    return  5
end

map([1, 2], fn (x) -> x * 2)

```

## If/else

```
if (1 > 2) 1 end

if (1 > 2)
    1
elif (2 > 2)
    2
else
    3
end
```

## Switch

```
case (1 > 2)
    true -> 1
    _ -> 2
end
```

## Pipeline operator

```
"hello" -> capitalize()
```

## Array

```
let a = [1, 2, 3]
```

## Iterators

```
map(a, fn (x) -> x)
map(a, fn (x)
    x
end)
```
